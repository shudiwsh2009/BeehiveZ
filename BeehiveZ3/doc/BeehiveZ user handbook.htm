2.2  index management

You can click the cell of state to use some index or unuse some index.

The existing indexes are as follows.

[1]       Petri net index

1)         LengthOnePathIndex: for exact Petri net retrieval, only index on length one pathes

2)         LengthTwoClosurePathIndex: for exact Petri net retrieval, index on length one and two pathes

3)         NullIndex: for exact Petri net retrieval, no index for sub-graph query

4)         TARLuceneIndex: for similar Petri net retrieval, index on the tars

5)         NoSimilarIndex: for similar Petri net retrieval, no index for similar query based on tars

6)         TaskRelationIndex: task relations index, can be used to query Petri nets based on ordering relations

[2]       YAWL index

1)         YAWLTasksLuceneIndex: YAWL tasks index, for exact YAWL model query

2)         NullYAWLIndex: for exact YAWL model query, no index for sub-graph query

4  Query language for TaskRelationIndex

²  <Expression> ::= <AndExpression> (<Or> <AndExpression>)*

²  <AndExpression> ::= (<RelationExpression> | <ExistExpression> | "("<Expression>")") (<And> (<NotExpression> | <RelationExpression> | <ExistExpression> | "("<Expression>")"))*

²  <NotExpression> ::= <Not> (<RelationExpression> | <ExistExpression> | "("<Expression>")")

²  <ExistExpression> ::= <Exist>(<Activity> | "("(<Activity>)+")")

²  <RelationExpression> ::= <Activity> <Relation> <Activity>

²  <Exist> ::= "exist"

²  <Activity> ::= "\"" (~["\""])+ "\""

²  <Relation> ::= (<ParallelWith> | <Exclude> | <Precede>)

²  <ParallelWith> ::= "parallel with" | "=="

²  <Exclude> ::= "exclude" | "##"

²  <Precede> ::= "precede" | "->"

²  <And> ::= "and" | "&&"

²  <Or> ::= "or" | "||"

²  <Not> ::= "not" | "!"
